# 06 Special Exporters

Examples for extending Prometheus monitoring with additional exporters.

## Files

- `blackbox-exporter-values.yaml`: Basic Blackbox Exporter Helm values.
- `blackbox-servicemonitor.yaml`: ServiceMonitor example for Blackbox Exporter.
- `nginx-metrics-values.yaml`: NGINX Ingress metrics values example.
- `dcgm-exporter-notes.md`: NVIDIA DCGM Exporter installation notes.
