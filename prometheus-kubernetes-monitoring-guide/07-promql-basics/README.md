# 07 PromQL Basics

Reusable PromQL examples for Kubernetes operations.

## Files

- `kubernetes-resource-queries.promql`: CPU and memory queries.
- `pod-status-queries.promql`: Pod status and restart queries.
- `http-service-queries.promql`: request, error, latency queries.
- `exporter-queries.promql`: Blackbox and exporter-oriented queries.
