# Operation Checklist

## Routing

- [ ] 쉬운 요청과 어려운 요청 기준이 명확한가?
- [ ] fallback 모델이 정의되어 있는가?
- [ ] 라우팅 결정 로그를 남기는가?
- [ ] 잘못 라우팅된 요청을 다시 분석할 수 있는가?

## Ollama

- [ ] Ollama 서버가 안정적으로 실행되는가?
- [ ] 모델 로딩 시간이 허용 가능한가?
- [ ] CPU/GPU/메모리 사용량을 모니터링하는가?
- [ ] 장애 시 Bedrock으로 fallback 가능한가?

## Bedrock

- [ ] 모델 접근 권한이 설정되어 있는가?
- [ ] 리전과 모델 ID가 명확한가?
- [ ] 토큰 사용량과 비용을 추적하는가?
- [ ] rate limit과 retry 정책이 있는가?

## Security

- [ ] 외부 사용자 입력을 그대로 신뢰하지 않는가?
- [ ] 프롬프트와 응답 로그에 민감정보가 남지 않는가?
- [ ] API 인증과 권한 관리를 별도로 설계했는가?
